# 開發日誌（Dev Session Logs）

由 **Harper（HR）** 在每個 Cursor 對話**結束前**撰寫，供日後回顧與新 Chat 銜接。

| 項目 | 說明 |
|------|------|
| **位置** | `deploy/dev-logs/YYYY-MM-DD-session-NN-<slug>.md` |
| **與 PROGRESS 分工** | `deploy/PROGRESS.md` = 技術交接與下一步；本目錄 = 當次對話摘要、決策、待你回覆的事項 |
| **團隊副本** | 鮪魚同意時，Harper 同步摘要至 `~/.cursor/team/meetings/` |
| **機密** | 勿寫 API Key、密碼、完整 connection string |

## 檔名

- `YYYY-MM-DD`：對話主要日期（跨日以最後更新日為準）
- `session-NN`：專案內對話序號（與鮪魚口頭「第 N 個對話」對齊）
- `<slug>`：簡短英文主題，如 `dify-cloud-deploy`

## 模板（Harper 填寫）

見同目錄任一 `session-*.md` 標題下的章節結構。
